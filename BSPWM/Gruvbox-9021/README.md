## After Install
* Copy contents .config in directrory ~/.config/
```cp -r * ~/.config```
* Make the files executable
```chmod +x -R ~/.config/ or chmod +x ~.config/desired\ file/```
* If you have xinitrc, also write this
```exec bpswm -c ~/.config/bspwm/bspwmrc ```
